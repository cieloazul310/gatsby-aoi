Gatsby Theme Aoi Blog は、**Gatsby Theme Aoi** で作成された [Gatsby スターター][Gatsby Starters]です。Gatsby Theme Aoi は [MUI] (Material-UI) で構成された [Gatsby テーマ][Gatsby Themes] です。Gatsby Theme Aoi には以下のような特徴があります。

- [Gatsby] v5、[MUI] v5 に対応
- [MUI] で構成されたシンプルなレイアウト
- [MUI] コンポーネントに [Gatsby のルーティング][Gatsby Link]を実装したコンポーネント
- [MUI] や [emotion] の設定不要
- ダークモードを実装
- サイトを包括したステートの実装

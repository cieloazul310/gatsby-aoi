Gatsby Theme Aoi では、MUI で構成されたシンプルなレイアウトコンポーネント (Aoi レイアウト) を利用できます。

### 基本的な使い方

`embed:ExampleCode-1.tsx`

Aoi レイアウトは `@cieloazul310/gatsby-theme-aoi` からインポートできます。

### Aoi レイアウトの制御

`embed:ExampleCode-2.tsx`

Aoi レイアウトに含まれる [Drawer](https://mui.com/components/drawers/)、[BottomNavigation](https://mui.com/components/bottom-navigation/)、[Fab](https://mui.com/components/floating-action-button/) は、props から表示を制御できます。

#### componentViewports

| key             | default    |
|-----------------|------------|
| swipeableDrawer | `'smDown'` |
| permanentDrawer | `'mdUp'`   |
| bottomNav       | `'xsDown'` |
| fab             | `'smDown'` |

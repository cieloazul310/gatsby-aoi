Gatsby Theme Aoi に含まれるパッケージを紹介します。

- `@cieloazul310/gatsby-theme-aoi`
- `@cieloazul310/gatsby-theme-aoi-components`
- `@cieloazul310/gatsby-theme-aoi-layout`
- `@cieloazul310/gatsby-theme-aoi-top-layout`
- `@cieloazul310/gatsby-theme-aoi-utils`

[hupe1980/gatsby-theme-material-ui](https://github.com/hupe1980/gatsby-theme-material-ui) のパッケージ構成を参考にしています。
これらのパッケージは個別にインストールが可能です。

```shell
yarn add @cieloazul310/gatsby-theme-aoi-components
```

```tsx
import { AppLink } from '@cieloazul310/gatsby-theme-aoi-components';
```

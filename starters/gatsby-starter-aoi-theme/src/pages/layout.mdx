Gatsby Theme Aoi Layout is a simple layout built with MUI including `<AppBar>`, `<Drawer>`, `<BottomNavigation>`, `<Fab>`.

```typescript
import { Layout } from '@cieloazul310/gatsby-theme-aoi';

function Page() {
  return (
    <Layout title="Page Title">
      {children}
    </Layout>
  );
}

export default Page;
```
